# Screen Map (iPhone)

## Tab Bar
1. Сегодня
2. Гардероб
3. Образы
4. План
5. Профиль

## Screen Details

### Сегодня
- Карточка погоды.
- Блок "рекомендованный образ".
- Кнопки: "Перемешать", "Надеть", "Собрать вручную".

### Гардероб
- Сетка вещей.
- Фильтры: категория, цвет, сезон, формальность, статус.
- Поиск.
- Плавающая кнопка "Добавить".

### Добавить вещь
- Камера/галерея.
- Превью фото.
- Поля: категория, цвет, сезон, формальность, теги, статус.
- Кнопка "Сохранить".

### Образы
- Сохраненные образы.
- CTA "Новый образ".
- Фильтр: сезон/случай/избранное.

### Конструктор образа
- Слоты: верх, низ, обувь, слой, аксессуары.
- Панель доступных вещей.
- Валидация совместимости.
- Кнопки: "Сохранить", "Назначить в план".

### План
- Недельный календарь.
- Ячейка дня с образом.
- Быстрое изменение/удаление.

### Профиль
- Метрики носки.
- Неношеные вещи.
- Настройки приватности и Face ID.
