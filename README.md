# WardrobeFlow iPhone MVP

Этот репозиторий содержит:
- продуктовый пакет в папке `docs`
- рабочий MVP-каркас iPhone-приложения в папке `WardrobeMVP`

## Что реализовано
- Локальное хранение на `SwiftData` для вещей, образов и планов.
- Экран каталога с фильтрами, поиском и сменой статуса вещей.
- Добавление вещи с фото из галереи и камеры.
- Конструктор образов с бизнес-валидацией (верх + низ + обувь).
- Планировщик образов по дням.
- Экран "Today" с рекомендациями на основе погоды.
- Погодный сервис через Open-Meteo API.
- Профиль с аналитикой по неношеным вещам.

## Как запустить
1. Создайте новый iOS App проект в Xcode (`SwiftUI`, `Swift`, iOS 17+).
2. Добавьте все файлы из папки `WardrobeMVP` в target проекта.
3. Убедитесь, что стартовый файл приложения - `WardrobeMVPApp.swift`.
4. Добавьте `NSCameraUsageDescription`, `NSPhotoLibraryUsageDescription`, `NSLocationWhenInUseUsageDescription` в `Info.plist`.
5. Запустите на симуляторе или физическом iPhone.
